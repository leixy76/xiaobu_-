### **预测加工理论核心词汇表 (Glossary for Predictive Processing Theory)**

#### **1. 核心理论与框架 (Core Theories & Frameworks)**

| 英文术语 (English Term) | 中文翻译 (Chinese Translation) | 严谨释义 (Rigorous Definition) |
| :--- | :--- | :--- |
| **Predictive Processing (PP)** | 预测加工 (Yùcè Jiāgōng) | 一种认知科学和神经科学的广义理论框架，主张大脑并非被动接收感官信息，而是主动地、自上而下地生成关于世界（包括身体）的预测，并根据自下而上的预测误差（prediction error）来不断更新这些预测，从而推断感觉输入的隐藏原因。该框架试图统一解释知觉、行动、学习和认知。 |
| **Free Energy Principle (FEP)** | 自由能原理 (Zìyóu Néng Yuánlǐ) | 由卡尔·弗里斯顿（Karl Friston）提出的一个根本性原理，旨在为生命和心智提供一个统一的理论。它假定任何自组织系统（如生物体）为了维持其存在，必须最小化其变分自由能（variational free energy）。这在信息论上等同于最小化其感官状态的“惊异”（surprise）或预测误差的长期平均值。 |
| **Active Inference** | 主动推理 (Zhǔdòng Tuīlǐ) | 源于自由能原理的过程理论，解释了生物体如何通过行动和感知来最小化自由能/预测误差。它统一了行动（通过改变世界使之符合预测）和知觉（通过改变内在预测使之符合世界），认为两者都服务于最小化预测误差这同一个目标。 |
| **Predictive Mind** | 预测心智 (Yùcè Xīnzhì) | 一种认知科学范式，将心智或大脑的根本功能刻画为一个主动的、层级化的预测机器。在这种观点下，所有心智现象，从低阶知觉到高阶思维，都被理解为预测生成与误差修正的动态过程。 |

#### **2. 关键机制与概念 (Key Mechanisms & Concepts)**

| 英文术语 (English Term) | 中文翻译 (Chinese Translation) | 严谨释义 (Rigorous Definition) |
| :--- | :--- | :--- |
| **Prediction Error Minimization (PEM)** | 预测误差最小化 (Yùcè Wùchā Zuìxiǎohuà) | 预测加工框架的核心计算目标。指大脑不断减少其“自上而下”的预测与“自下而上”的实际感官输入之间的差异（即预测误差）的过程。该机制被认为是驱动学习、知觉修正和行动选择的根本动力。 |
| **Bayesian Inference / Bayesian Brain** | 贝叶斯推理 / 贝叶斯大脑 (Bèiyè sī Tuīlǐ / Bèiyè sī Dànǎo) | 将大脑建模为一个使用贝叶斯概率进行推理的统计机器。它描述了大脑如何结合先验信念（prior beliefs）和新的感官证据（likelihood）来计算和更新关于世界状态的后验概率（posterior probability）的过程，这是预测加工的数学基础。 |
| **Generative Model** | 生成模型 (Shēngchéng Móxíng) | 大脑内部的、关于世界（包括身体）如何产生感官信号的因果结构和统计规律的概率模型。大脑利用这个模型来生成对感官输入的预测。模型的准确性通过预测误差最小化的过程不断得到优化。 |
| **Hierarchical Predictive Coding** | 层级预测编码 (Céngjí Yùcè Biānmǎ) | 描述预测加工在神经层面如何实现的具体架构。在该架构中，大脑皮层被组织成一个层级结构，高层级向低层级发送预测信号，而低层级仅将未被高层级预测解释的残余信号——即预测误差——向前传递。这种编码策略在信息处理上极为高效。 |
| **Surprise (Information-theoretic)** | 惊异 (Jīngyì) | 一个信息论概念，指某个感官状态发生的负对数概率（negative log-probability）。概率越低的事件（即越“出乎意料”的事件）其惊异值越高。根据自由能原理，生物体必须通过最小化自由能来避免长时间处于高惊异的状态，以维持其生存。 |
| **Perception and Action** | 感知与行动 (Gǎnzhī yǔ Xíngdòng) | 在主动推理框架下，这是最小化自由能或预测误差的两种互补方式。**感知**是通过更新内在预测（信念）来减少误差（即改变心智以适应世界）；**行动**是通过改变感官采样来减少误差（即改变世界以适应心智）。 |
| **Homeostasis** | 内稳态 (Nèiwěntài) | 生物学概念，指生命系统维持其内部环境（如体温、血糖水平）相对稳定的能力。在预测加工理论中，维持内稳态被视为自由能最小化的一个核心生物学目标，有机体通过预测和调节内感受（interoception）信号来实现。 |
| **Markov Blanket** | 马尔可夫毛毯 (Mǎ'ěrkěfū Máotǎn) | 一个统计学上的边界概念，用于在一个系统中将内部状态（internal states）与外部状态（external states）分离开。这个边界由感觉状态（sensory states）和主动状态（active states）构成，定义了有机体与环境进行信息交换的接口。 |
| **Precision / Precision Weighting** | 精度 / 精度加权 (Jīngdù / Jīngdù Jiāquán) | 指对预测误差信号的可靠性或置信度的估计。大脑会根据情境（如环境的确定性）为不同的预测误差分配不同的精度权重。高精度的误差信号会对信念更新产生更大的影响。这一机制被认为是注意（attention）的神经计算基础。 |

#### **3. 相关哲学与认知概念 (Related Philosophical & Cognitive Concepts)**

| 英文术语 (English Term) | 中文翻译 (Chinese Translation) | 严谨释义 (Rigorous Definition) |
| :--- | :--- | :--- |
| **Embodied Cognition** | 具身认知 (Jùshēn Rènzhī) | 一种认知科学观点，强调认知过程深深植根于身体的形态、感觉运动能力及其与环境的动态交互中。行动导向的预测加工模型（action-oriented PP）与具身认知观点高度契合。 |
| **Enactivism** | 生成主义 (Shēngchéng Zhǔyì) | 一种激进的具身认知观点，认为认知并非对外部世界的内在表征，而是通过有机体与环境之间的动态耦合“生成”（enact）的。预测加工理论为这种“生成”过程提供了可能的机制性解释，但两者在“表征”（representation）问题上存在张力。 |
| **Representation** | 表征 (Biǎozhēng) | 心灵哲学中的核心概念，指心智中代表或“关于”世界某些方面的状态或过程。在预测加工理论内部存在争论：一些版本认为生成模型是丰富的内在表征；而另一些行动导向的版本则认为表征是稀疏的、服务于行动的，甚至可以被取消。 |
| **Cognitive Penetrability** | 认知可渗透性 (Rènzhī Kě Shèntòu Xìng) | 指高阶认知状态（如信念、欲望、情绪）能否以一种语义连贯的方式直接影响低阶知觉经验的现象。预测加工的自上而下特性天然地支持知觉是认知可渗透的。 |
| **Affordance** | 可供性 (Kě Gōng Xìng) | 源自生态心理学的概念，指环境为特定行动者提供的行动可能性（例如，椅子“提供”了“坐”的可能性）。在预测加工模型中，对可供性的知觉被认为是指导行动选择的关键。 |
| **The Bounds of Cognition** | 认知边界 (Rènzhī Biānjiè) | 关于认知过程发生在哪里的哲学问题。内在主义版本的预测加工将认知边界限定在马尔可夫毛毯之内（通常是大脑或身体）；而与延展认知（extended cognition）结合的版本则认为认知可以延展到外部环境中。 |

#### **4. 应用领域与议题 (Application Fields & Issues)**

| 英文术语 (English Term) | 中文翻译 (Chinese Translation) | 严谨释义 (Rigorous Definition) |
| :--- | :--- | :--- |
| **Social Cognition / Social Mind** | 社会认知 / 社会心智 (Shèhuì Rènzhī / Shèhuì Xīnzhì) | 指个体处理和运用关于他人信息的能力。在预测加工框架下，社会认知被理解为两个（或多个）预测大脑之间的双向预测过程，双方都试图通过观察对方的行为来最小化对自己预测的误差，从而推断对方的意图和信念。 |
| **Mindreading** | 读心 (Dúxīn) | 将信念、欲望、意图等心理状态归因于他人的能力。预测加工为读心提供了一个基于推断（inference）的统一模型，认为我们通过使用与理解自身相似的生成模型来预测和解释他人的行为。 |
| **Emotion** | 情绪 (Qíngxù) | 在预测加工模型中，情绪通常被理解为对内感受信号（interoceptive signals）的预测和调控。例如，焦虑可能被建模为对身体唤醒状态的预测精度过高，导致对轻微的身体变化过度反应。 |
| **Culture / Enculturation** | 文化 / 文化濡化 (Wénhuà / Wénhuà Rúhuà) | 文化被视为一组共享的先验信念和生成模型，通过文化濡化过程被个体习得。这些共享的预测塑造了个体的知觉、行动和社会互动模式，从而解释了认知的文化特异性。 |
| **Human-Machine Interaction (HMI)** | 人机交互 (Rén-Jī Jiāohù) | 预测加工和主动推理框架被用于建模人与智能体（如机器人）之间的信任和协作。信任的建立被看作是一个双向预测过程，人类用户和机器人都试图预测对方的行为以实现共同的目标。 |
